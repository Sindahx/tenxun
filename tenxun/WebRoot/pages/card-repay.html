  1. 1

    获取微信财付通帐号

    UE财付通 -> 支付管理 -> 微信支付 -> 微信支付帐号，输入用户手机/QQ查询，并核对用户姓名

  2. 2

    获取微信字母开头帐号

    打开wxpay.oa.com -> 客服查询 -> 账户查询 -> 微信账户查询，查询方式：acctid （微信财付通帐号去掉后缀@wx.tenpay.com）

  3. 3

    查询还款记录 - 非实时到账

    UE财付通 -> 支付管理 -> 微信支付 -> 微信信用卡还款，输入微信字母开头帐号，选择还款日期

    若超时未到账，显示还款中，到账时间为 "非实时到账" ，复制“财付通提现单号” 前往 “提现记录查询” 再次确认

  4. 4

    查询还款记录 - 实时到账

    若超时未到账，显示还款中，到账时间为 "实时到账" ，复制“财付通提现单号” 前往 “实时还款查询” 再次确认

    END

  2. 2

    微信支付还款失败资金何时退回

    零钱支付，金额会从还款失败后1个工作日内退回零钱。
    银行卡支付，金额会从还款失败后3个工作日内退银行卡。

  3. 3

    还款成功，但用户表示未收到，如何处理

    尊敬的用户您好，关于您反馈的问题，请您进入钱包点击右上角“…”=》“帮助中心”=》“其他问题”=》“信用卡还款”=》“信用卡还款超时未到账怎么办”=》“联系客服”，根据页面提示提交相关资料，我们会尽快为您核实处理，感谢您的支持与理解，谢谢！
  4. 4

    信用卡还款失败产生滞纳金如何处理

    尊敬的用户您好，关于您反馈的问题，请您进入钱包点击右上角“…”=》“帮助中心”=》“其他问题”=》“信用卡还款”=》“【信用卡还款】产生滞纳金如何处理”=》“联系客服”，根据页面提示提交相关资料，我们会尽快为您核实处理，感谢您的支持与理解，谢谢！

  1. 1

    财付通客户端或PC端还款

    UE财付通 -> 交易管理 -> 交易查询 -> 提现记录查询(新)，输入财付通帐号，选择还款日期

  2. 2

    手机QQ钱包还款

    UE财付通 -> 支付管理 -> 手Q支付 -> 手Q还款查询，输入财付通帐号，选择还款日期

  3. 3

    不管客户端、PC端、手Q端，均可使用提现单号，在提现记录查询还款情况

    UE财付通 -> 交易管理 -> 交易查询 -> 提现记录查询(新)，输入提现单号，选择还款日期

    END

    还款失败，金额会在3个工作日内退回余额.

  3. 3

    还款成功，但用户表示未收到，如何处理

    您好，关于您反馈的问题，请您登录网址：kf.qq.com 联系财付通在线客服，提交信用卡出入账明细记录截图，感谢您的理解与支持，谢谢！
  4. 4

    信用卡还款失败产生滞纳金如何处理

    您好，关于您反馈的问题，请您先不要担心，烦请您先把产生的费用自行还上，并登录kf.qq.com联系财付通在线客服提供以下资料：财付通账号、还款时间、还款金额、银行卡信息、还款到期日，产生滞纳金金额，及产生滞纳金的网上截图明细（这个是需要登录您的网上银行明细中查询并提供截图的，若是没有开通网银，烦请您尽快联系银行的工作人员提供相关截图后再给到我们）。由于需要与银行确认查询核实，预计10个工作日内会有处理结果，请您耐心等待，谢谢！
