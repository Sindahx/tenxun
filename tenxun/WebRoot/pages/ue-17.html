# 退款单查询

查询方法

# 交易管理 - 交易查询 - 退款单查询

1、退款成功到零钱：1217959901341508287346284807

2、退款成功到银行卡：1231518002351509128375290530

3、转入代发：1216845401351507178305890925

4、退款失败转退余额：1217834101321508227063898089

退款类型

1）B2C退款 = 退回零钱
2）银行直接退款/提现退款 = 退回原支付银行卡

退款状态

1）退款流程中：已发起退款申请，未到账
2）退款成功：退款已成功到账
3）转入代发：退款失败，资金已返回商家，引导联系商家通过其他渠道办理退款
4）退款失败转退余额：退回银行卡失败，转退零钱

退款时间及路径

# 充值状态失败
快捷交易掉单 充值状态：“失败”，交易状态：“等待买家支付”，1-3个工作日原路退回
充值状态：“失败”，交易状态：“交易关闭”，1-5个工作日原路退回
重复扣费 充值状态：“失败”，1-3个工作日原路退回
# 充值状态成功
大部分订单交易 1-3个工作日原路退回
转账 收款方超过24小时未收取的资金或拒收的资金将 退回零钱
红包 超过24小时未发送或未被领完，将 退回零钱。
信用卡还款 （非实时到账）还款失败，资金会在失败后 1个工作日内退回零钱。
（实时到账，零钱支付）还款失败，资金会在失败后 1个工作日内退回零钱
（实时到账，银行卡支付）还款失败，资金都将会在失败后 3天内退回零钱。
话费充值 储蓄卡/零钱支付的将在 6小时 内退款到零钱；信用卡支付的将在 2-5 个工作日内退款到信用卡。
付款（以前叫刷卡） 零钱退款当天内到账，银行卡退款两个工作日到账
# 订单异常
红包 1、支付成功，未被领取，已超过24小时未退款，引导等待5个工作日留意退款。
2、若核实到交易时间非最近五天，未被领取但超时未退款，升级至微信平台运营组
http://kbs.cm.com/v3/knowledge/show/150902uI7vmi150902amYB3Y/803108121
转账 1、支付成功，卖家真实名称为“微信转账中转账户+数字”
2、微信支付运营系统查询订单编码无记录，若未超过24小时，引导“重发消息”
3、若无法重发或超过24小时未退回，引导等待5个工作日退款
4、若超过5个工作日未退，升级微信平台运营组
http://kbs.cm.com/v3/knowledge/show/150915jueEry150915rAF3yM/803108121
# 提现
零钱提现 零钱提现失败，资金立即退回零钱
理财通提现 “货币基金/保险理财”赎回失败退回对应基金账户，“定期基金/指数基金”赎回失败退回理财通余额

1）使用银行卡第三方商户交易退款，在退款过程中，由于原卡注销/挂失/失效等，导致原卡无法接收退款的资金
转入代发：证明退款失败，资金已返回商家，引导联系商家通过其他渠道办理退款

2）使用银行卡支付掉单，在退款过程中，由于原卡注销/挂失/失效等，导致原卡无法接收退款的资金
