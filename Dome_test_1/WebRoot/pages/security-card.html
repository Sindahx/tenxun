# 更换流程
  1. 1

    总资产 < 2000元，不管旧安全卡是否正常

    总资产<=5元，验证新预留手机；总资产>5元，验证旧预留手机，如旧预留手机无法使用，请走第三步，提交资料

    您好，针对您账户的情况，建议您先绑定需要更换的银行卡，然后打开理财通界面=》选择“我的”=》进入头像下方的“安全卡”=》点击橙色“更换安全卡”=》选择“开始换卡”=》选择想要更换的安全卡=》短信验证后即可更换成功。
  2. 2

    总资产 >= 2000元，且旧安全卡正常

    1、登录微信客户端，进入“钱包”=》“银行卡”=》“添加银行卡”=》绑定新银行卡；
    2、进入理财通，将理财通账户的所有资金取出；
    3、取出成功后，大约需要1天，再用新银行卡进行买入；
    4、买入成功即更改理财通安全卡成功。
    (理财通资金取出后剩余资金小于5元时，验证新绑卡预留的手机号即可更换安全卡。)
  3. 3

    总资产 >= 2000元，但旧安全卡遗失/注销

    尊敬的用户您好，如需更换安全卡，请您按以下方式操作：请绑定支持购买理财通的新银行卡，绑定成功后：打开理财通=》我的 =》安全卡=》更换安全卡=》选择“当前理财通安全卡已遗失或注销”=》资料准备好了，更换安全卡=》并按页面提示提交资料，我们的工作人员在收到资料后会在2个工作日内处理。

    需准备的资料有：
    1) 旧银行卡注销证明或挂失证明复印件（需盖银行公章）；
    2) 新银行卡卡号
    3) 手持身份证照片（新安全卡为招行卡时必须提供）。
    注：如无法提供注销/挂失证明，请提供银行打印的旧卡资金流水证明（须银行公章），且包含一笔财付通资金流水记录。若这两项均无法提供，请补充手持身份证照片
    END
