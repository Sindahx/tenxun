# 交易记录查询

用户查询交易单号方法

微信钱包 =》右上角“…” =》交易记录

交易单号分类

① 我司业务，例如：红包、转账、信用卡还款、Q币充值，10开头纯数字

② 第三方商户交易，例如：友宝、话费充值、大众点评，滴滴打车，12开头，28位纯数字

后台查询交易单号方法

① 若使用零钱支付，则在风控管理 - 资金流水，通过 C 帐号查询 或 在KFWP - 账户资料 - 资金流水查询

例如：查询C帐号 085e9858ee5ad8eab5a6ec1c1@wx.tenpay.com ，2015-09-24 14:14:59 使用零钱发送1.99元红包的交易单号

② 若使用银行卡支付，则在交易管理 - 充值记录，通过 C 帐号查询 或 在KFWP - 交易查询 - 充值查询

例如：查询C帐号 085e9858ee5ad8eab5a6ec1c1@wx.tenpay.com ，2015-09-23 02:57:54 使用银行卡发送99.99元红包的交易单号

后台交易记录查询方法

② 第三方商户交易：1219772901201509152147339163

③ 微信话费充值：1217189101331507206633382510

字段说明

买家账户号码：付款方微信财付通帐号

卖家真实姓名：商家名称（若显示腾讯公司，则为我司业务或我司渠道充值的话费）

支付通支付：零钱支付

卖家真实姓名：商家名称（若显示腾讯公司，则为我司业务或我司渠道充值的话费）

交易状态：支付成功/等待卖家发货、交易关闭、转入退款、等待付款、空白

基金基础代扣：即自动续费业务 或 委托代扣（免密支付）
